# DING0 Workshop

Material from (Mobi-)DING0-Workshop, 23.08.2022.

## Installation

### Windows

    conda env create -f ding0_workshop_env.yml
    conda activate ding0_workshop_env

### Proper operating system

As above or see https://dingo.readthedocs.io
